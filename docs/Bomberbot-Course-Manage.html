BOMBERBOT

Empowering all girls and boys around the globe to develop digital skills.

Bomberbot is an education platform designed to help kids of different ages to develop skills that are taking force in this new era, the era of digitalization.

Offers three different programs in which students develop basic knowledge about programming and computational thinking through lessons, games and bootcamps.

Features

CONTENT MANAGEMENT SYSTEM

Data Visualization

The tree viewer makes it easy for the teacher to navigate and to see which blocks belong to which course, which lessons belong to which block and which activities belong to which lesson.

Using the "Click vs Click" mode per lesson to see which lessons belong to a particular course. 

Going from 6 steps to 3 steps per block to know the course’s lessons.

Control panel

With the options to create, modify and delete content using the "click vs click" mode. To create a course with a lesson block with an activity.

Going from 20 steps to 7 steps to create a complete course

Automatic realations

The teacher can relate the different elements automatically by creating them in their respective parent. Using the click vs click mode creates three lessons and can relate them to the same block.

Going from 10 clicks to 4 clicks to Create a lesson and relate it to an existing block in a course.
